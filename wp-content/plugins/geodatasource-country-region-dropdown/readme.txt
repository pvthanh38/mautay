== Description ==

GeoDataSource Country Region DropDown plugin use the shortcode to display the country and region dropdown. A dropdown menu created with javascript that allows user to implement country-region dropdown list effortlessly. Country-Region Dropdown Menu is needed mostly in forms where the region field will get updated automatically when there is a change on the country field.

It can be applied as easy as by entering the given required shortcode into the post, article, page or Contact Form. For example,`[gds-country-dropdown]`

== Installation ==

= Manual Installation (Use this method only if you are unable to install the plugin via the admin area) =
1. Download the plugin zip file.
2. Unzip and upload the `geodatasource-country-region-dropdown` folder to the `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. You can now start using GeoDataSource Country Region DropDown with the shortcodes provided.
